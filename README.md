1. **Clone the repository:**
    ```bash
   git clone https://github.com/GDSC-MSCW/GDSCMSCWWebsite.git
2. Navigate to the project folder:
   ```bash
   cd C:\Users\example\GDSCMSCWWebsite

3. Install dependencies:
   ```bash
    npm install

   ```bash
   npm run dev
